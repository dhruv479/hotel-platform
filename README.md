# Hotel Booking App

## Installation

```bash
$ npm install
```

## Environment Variables
### Local 
  Copy `.env.example` to `.env` and update the values accordingly.
### Prod
  Reference variables from `.env.example` and set secrets in the deployment environment.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
